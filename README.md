## 特性

### 基础调整

1. 可以吃素食，但是收益减半（该选项可配置）
2. 吃肉时根据溢出的饱食度增加灵感值

### 奔雷矛

1. 改良：增加暗影心房作为升级材料
2. 新增：充能奔雷矛可以使用充电火花柜/附身暗影心房升级成武神长矛
3. 新增：武神长矛拥有 68 攻击力、30 位面攻击力、25% 移速；新增技能冲天刺，cd 3 秒
4. 新增：冲天刺落地 2 秒内获得高额防御，冲天刺可以快速上下牛
5. 改良：普通奔雷矛的冲刺也可以恢复耐久度
6. 改良：普通奔雷矛增加 15% 移速
7. 改良：使用 1 个电羊角可以将战斗长矛升级成奔雷矛
8. 新增：战斗长矛可投掷

### 盾牌

1. 改良：取消盾牌装备时的 CD
2. 改良：成功格挡后立刻重置格挡 CD
3. 改良：盾牌格挡后的攻击可以恢复 16 点耐久度
4. 改良：攻击会嘲讽目标 3 秒

### 训牛

1. 改良：骑牛技能 3 的灵感值可自动恢复上限从 70 提升至 100
2. 改良：骑牛技能 2 的骑乘时间从 30% 提升至 300%
3. 新增：骑乘攻击可以治疗牛牛
4. 新增：骑乘时可以使用武器

### 头盔

1. 改良：生命值不满时也可以恢复统帅头盔耐久度
2. 新增：使用启迪碎片可以将统帅头盔升级为武神头盔
3. 新增：武神头盔拥有 85% 防御、90% 防水，并根据敌人的血量造成额外伤害
4. 改良：使用 1 个大理石可以将战斗头盔升级成统帅头盔

### 乐谱

1. 改良：武器颤音额外效果：颤音效果大幅提升
2. 改良：心碎歌谣额外效果：不会睡着
3. 改良：醍醐灌顶额外效果：召唤一道光
4. 改良：英勇赞歌额外效果：不被风暴影响。
5. 改良：防火假声额外效果：攻击使对方潮湿
6. 改良：启迪摇篮额外效果：不被虚影攻击
7. 改良：黑暗悲歌额外效果：不被影怪攻击
8. 改良：粗鲁插曲额外效果：嘲讽维持 3 秒、破防，同时对 BOSS 也会生效
9. 改良：惊心独白额外效果：降低移速
10. 改良：战士重奏额外效果：为没死的队友回血
11. 新增：狂想曲，可以修复手中的武器。（强度偏高，默认关闭，如有需要可自行开启）
12. 战斗筒子不会自动关闭，并修改了 UI


## Bug 反馈

如果遇到崩溃、报错或功能异常，可以提交 Issue，并尽量带上以下信息：

1. 问题发生时正在做什么，以及是否可以稳定复现
2. 本模组的配置选项，以及是否开了其他模组
3. 问题发生后的日志文件

日志文件位置：

- 客户端日志：`文档\Klei\DoNotStarveTogether\client_log.txt`
- 地面服务器日志：`文档\Klei\DoNotStarveTogether\master_server_log.txt`
- 洞穴服务器日志：`文档\Klei\DoNotStarveTogether\caves_server_log.txt`

如果服务端放在专服或者云服上，服务器日志的位置如下：
- 地面服务器日志：`<服务器存档位置>\Cluster_<集群名>\Master\server_log.txt`
- 洞穴服务器日志：`<服务器存档位置>\Cluster_<集群名>\Caves\server_log.txt`
